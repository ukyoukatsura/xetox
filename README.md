# XETOX

Deep Learningで競馬の予想をする。

## Description

Pytorchを使用し、簡単な全結合層で競馬の予想をする。

## Features

- 競走馬が1位になる確率を出力する。

## Requirement

- Pytorch
- GPU

## Installation

```
pip install pytorch numpy
git clone https://github.com/ukyoukatsura/xetox.git
```

## Usage

```
Python xetox.py
```
